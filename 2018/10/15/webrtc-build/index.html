WebRTC-Streamer 빌드 방법

윈도우 환경에서 빌드를 위해서는 Visual Studio 가 설치되어 있어야 한다. WebRTC 최신버전에서는 VS2017까지 지원하고 있지만, 하위호환을 위해서 VS2015로 진행해야 한다.

Depot Tools 설치

  • 환경변수 PATH 최상단에 depot_tools가 설치되어 있는 디렉토리 추가

  • gclient 실행

Windows SDK 및 Visual C++ 관련 패키지 설치

  • Visualstudio 2017 Community에서 아래 항목이 설치가 되어 있는지 확인해서 설치가 되어 있지 않다면 설치한다.
    • Visual C++ -> Common Tools for Visual C++ 2017
    • Microsoft Foundation Classes for C++
    • Windows XP Support for C++
  • Windows 10 SDK 설치 [1]
    • Debugging Tools for Windows 는 꼭 설치해야 한다.
